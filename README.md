## Installation

Clone the repo and install as a Claude Code plugin:


```bash
git clone https://github.com/kalibrio/atomic-scaling-os.git ~/.claude/plugins/atomic-scaling-os && cd ~/.claude/plugins/atomic-scaling-os && claude /plugin install
```

Or install directly in Claude Code:

```
/plugin install github:kalibrio/atomic-scaling-os
```

## How It Works

The agents run on **scheduled autonomous loops**. You only step in for strategic decisions.

| Frequency | Agents | What happens |
|--|--|--|
| **Daily** | hypothesis-engine, revenue-optimizer | Experiment scan, LVV /CAC monitoring |
| **Weekly** | culture-pulse, rhythm-keeper, growth-tracker, love-machine, hypothesis-engine, forecast-analyst | Colture check, scorecards, metrics digest, iteration review |
| **Bi-weekly** | funnel-builder, love-machine | Funnel conversion audit, editorial calendar |
| **Monthly** | freemium-architect, forecast-analyst, community-engine, revenue-optimizer | Segment analysis, calibration scoring, revenue deep review |
| **Quarterly** | team-architect, mission-designer, community-engine | Org review, Merlin Exercise, community health report |

All agents output to **email digest**, **Discord** (pillar-specific channels), and **Google Sheets** (dashboards).

## Quick Start

1. **Take the assessment** at [AtomicScaling.com](https://atomicscaling.com) to identify which pillars need attention
2. **Start with your weakest pillar**  ÔÇö run the relevant agent
3. **Set up the weekly rhythm** with `/playbook-rhythm-keeper`
4. **Enable scheduled routines** for autonomous operation
5. **Review decision surfaces** ÔÇö the agents surface choices, you make the calls
